# Quinfall Item Database

Bu depo, **Quinfall** oyununda kullanılan tüm item, materyal, NPC, görev ve diğer oyun içi verilerin JSON formatında tutulduğu merkezi veri tabanını içerir.  
Amaç, oyun içi öğelerin düzenli, kolay erişilebilir ve geliştirilebilir bir yapıda saklanmasını sağlamaktır.

This repository contains the centralized database for **Quinfall**, storing all items, materials, NPCs, quests, and other in-game data in JSON format.  
The goal is to keep game assets organized, accessible, and easy to maintain.

---

## 📂 Dizin Yapısı / Directory Structure

Depoda bulunan bazı önemli JSON dosyaları / Some key JSON files in the repository:

- **items.json** → Genel item tanımları / General item definitions  
- **item_materials.json** → Craft ve üretim materyalleri / Crafting and production materials  
- **item_pets.json** → Evcil hayvan verileri / Pet data  
- **item_binek.json** → Binek (mount) verileri / Mount data  
- **item_prop.json** → Oyun içi props/objeler / In-game props  
- **skiller.json** → Skill tanımları / Skill definitions  
- **moblar.json** → Canavar/mob verileri / Monster data  
- **npcler.json** → NPC tanımları / NPC definitions  
- **npc_saticilar.json** → Satıcı NPC listesi / Vendor NPC list  
- **npc_token_takascilar.json** → Token takas NPC listesi / Token exchange NPC list  
- **klan_quest_liste.json** → Klan görevleri / Clan quests  
- **klan_savasi_oduller.json** → Klan savaşı ödülleri / Clan war rewards  
- **ships_objeleri.json** → Gemi objeleri / Ship objects  
- **powers_degerleri.json** → Güç değerleri / Power values  
- **setbonus_degerleri.json** → Set bonusları / Set bonuses  
- **rng_tanrisi_liste.json** → RNG Tanrısı listesi / RNG God list

## 🌐 Dil Desteği (Localization)

Oyun içi metinlerin farklı dil karşılıklarını bulmak için şu dizini kullanabilirsiniz:  
You can find localization files for different languages in the following directory:

The Quinfall\TheQuinfall_Data\StreamingAssets\LANGUAGES


Bu dizin, tüm `loc` karşılıklarını içerir ve JSON dosyalarındaki referanslarla eşleştirilir.  
This directory contains all `loc` references and matches them with JSON definitions.
